Stonks documentation

Stonks is a lightweight, python package for Stock Data. It is open source, and allows members of the community to contribute.

Features

Fast.

By Implementing Multiprocessing and Multi-threading, Stonks can take existing packages, and run them nearly twice as fast.

Lightweight.

Based off existing packages, Stonks serves as an easy to use wrapper that joins multiple packages and apis together, making your life easier.

For every project.

Stonks is very easy to implement, and can be used in virtually any Python Script, Project, or Notebook

Automatic.

Stonks can quickly scale up to your computing needs, and is very flexible.

Full search.

Our docs allow to quickly search and pinpoint what to you want to see.

About

Stonks is the brainchild of Dev Patel. He decided to make it with the goal of having one central package to handle most financial data analytics requests.
